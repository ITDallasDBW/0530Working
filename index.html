  • Position: Static, Relative, Absolute
    • Static (default): Follows normal flow of the page
    • Relative
      • Same as Static BUT:
      • Allows position: top, left, bottom, right
    • Absolute
      • Does NOT follow the normal flow of the page
      • ALlows position with top, left, bottom, right
      • Position is based on closest 'position:relative' parent
